Webflow CMS tokens. Replace the placeholder tokens below ({{Title}}, {{Meta Description}}, etc.) with the actual Webflow CMS field references for your blog collection. 7 · IndexNow — instant Bing / Copilot indexing 30 min

IndexNow is a free protocol from Microsoft (also used by Yandex, Naver, Seznam) that lets you ping search engines the moment content publishes — indexes within minutes instead of days. Critical for Lunar because Bing powers ChatGPT Search and Microsoft Copilot — faster Bing indexing = faster AI citation refresh.

Setup

  1. Generate a key: create a 32-character hex string (any random generator). For example: 8a3f9b2e7c1d6a5e4f8b9c2d1e3f5a7b
  2. Host the key verification file: create a Webflow page at /{key}.txt (e.g. /8a3f9b2e7c1d6a5e4f8b9c2d1e3f5a7b.txt) containing only that same key as plain text. This proves you own the domain.
  3. Verify in Bing Webmaster: Bing Webmaster → IndexNow tab → enter the key and the key file location. Verification is automatic.
  4. Wire up auto-submission via Webflow + Zapier (or Make):
    • Trigger: "Webflow: New CMS Item Published" (Blog Posts collection)
    • Action: HTTP POST to https://api.indexnow.org/indexnow with body:
      IndexNow POST payload
      {
  "host": "www.lunarstrategy.com",
  "key": "YOUR_32_CHAR_KEY_HERE",
  "keyLocation": "https://www.lunarstrategy.com/YOUR_32_CHAR_KEY_HERE.txt",
  "urlList": [
    "https://www.lunarstrategy.com/article/{{trigger.slug}}"
  ]
}
Verify: after publishing a new blog post, Zapier's run log should show 200 OK from api.indexnow.org. Bing Webmaster → IndexNow tab will show submitted URLs and crawl status.
