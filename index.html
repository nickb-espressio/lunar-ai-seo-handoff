5 · Article + BreadcrumbList on blog template 10 min Biggest leverage

Blog posts currently have zero structured data. Adding Article + BreadcrumbList gives them rich snippets in Google search (lifts CTR 15–30%) and gives AI engines clean signals (headline, author, date) to cite.

Webflow Designer → Pages panel (left sidebar) → find the Blog Posts CMS Collection template page (the page that starts with the dynamic icon, usually at article/[slug]) → click the gear icon on the template → scroll to Custom Code → Inside <head> tag → paste the block below → Save and Publish
Webflow CMS tokens. Replace the placeholder tokens below ({{Title}}, {{Meta Description}}, etc.) with the actual Webflow CMS field references for your blog collection. Use the "Add field" picker (looks like a + button) inside the custom code editor — that inserts the correct token syntax for each field. Common field names: Title, Meta Description, Main Image, Published Date, Author Name, Slug. Adapt to whatever your collection calls them.
HTML <script> — paste into Blog Posts template page → Inside <head> tag 
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "{{Title}}",
  "description": "{{Meta Description}}",
  "image": "{{Main Image url}}",
  "datePublished": "{{Published Date}}",
  "dateModified": "{{Updated Date}}",
  "author": {
    "@type": "Person",
    "name": "{{Author Name}}"
  },
  "publisher": {
    "@type": "Organization",
    "name": "Lunar Strategy",
    "logo": {
      "@type": "ImageObject",
      "url": "https://cdn.prod.website-files.com/684807441792526c9388329c/6848195a5531260e598aedaf_Logo.svg"
    }
  },
  "mainEntityOfPage": {
    "@type": "WebPage",
    "@id": "https://www.lunarstrategy.com/article/{{Slug}}"
  },
  "inLanguage": "en"
}
</script>

<script type="application/ld+json">
